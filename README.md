# 📝 React Todo App

A simple and responsive Todo App built using **React** and styled with **Tailwind CSS**.
This app allows users to manage daily tasks efficiently with persistent storage using localStorage.

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 🧹 Clear all tasks
* 💾 Data persistence using localStorage

---

## 🛠 Tech Stack

* React
* JavaScript
* Tailwind CSS
* Vite

---

## 🎨 UI Highlights

* Responsive design using Tailwind CSS
* Clean and modern layout
* Interactive task controls

---

## 📦 Run Locally

```bash
git clone https://github.com/prasanta-dev/react-todo-app.git
cd react-todo-app
npm install
npm run dev
```
